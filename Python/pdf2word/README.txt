To convert pdf to docx file: 
1 - Enter: pip install pdf2docx
2 - Run the python script with command python3 pdf2docx.py "input_file" "output_file"
    You can use name the output file however you like
    Make sure the python file and pdf file are in the same directory for shorter command
    Or you can enter the path of input file instead
3 - Run pdf2docx convert "input_file"
